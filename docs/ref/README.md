# 呼叫中心系统

## 项目简介
这是一个基于Java的呼叫中心系统，使用JDK21和SpringBoot3构建，主要用于对接FreeSWITCH提供完整的呼叫中心解决方案。

## 技术栈
- **Java**: JDK 21
- **框架**: Spring Boot 3.x
- **通信**: FreeSWITCH ESL (Embedded Script Language)
- **架构**: 纯API服务，无前端页面

## 核心功能

### 数据管理
- **租户管理**: 多租户架构支持
- **座席管理**: 座席状态、权限、技能组管理
- **分机管理**: 分机配置、状态监控
- **话单管理**: 通话记录、计费统计

### 呼叫功能
- **预览外呼**: 座席预览拨号功能
- **IVR系统**: 交互式语音应答配置
- **呼叫路由**: 智能呼叫分配算法
- **呼叫转接**: 坐席间呼叫转移
- **三方通话**: 多方会议功能

### 系统特性
- 实时呼叫状态监控
- 话务统计报表
- 呼叫录音管理
- 系统配置管理
- 监控告警

## 部署要求
- Java JDK 21+
- Spring Boot 3.x
- FreeSWITCH服务器
- 数据库支持

## API接口
本系统提供RESTful API接口，主要用于与FreeSWITCH进行通信，实现呼叫中心的核心业务功能。
